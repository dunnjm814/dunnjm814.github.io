What is mySampler?

An interactive web application bringing a minimalist audio workflow into the browser environment. This web applications was developed with React.js, Redux, Javascript, and CSS with a Python, Flask, SQLAlchemy backend, tied together with PostgreSQL as the database. Tone.Js and the web audio api were used for audio playback, audio scheduling, audio signal generation, and routing/manipulation. AWS S3 cloud storage was put to good use here to house all the users audio clips and send back a URL to store in the database.
